# Portal Runner (SLUS-20003)

This patch completely separates the gamea from its disc image, making all loading times nearly instant. Included is a pnach cheat file that contains its 60 FPS patch as well as a patch that enables the game's built-in HostFS flag. This shouldn't be needed after patching the executable, however I've included it just in case.

Additionally, if you use the debugging-related pnach cheats found on [the game's page on The Cutting Room Floor](https://tcrf.net/Portal_Runner_(PlayStation_2)), you can now modify values inside DATA/VIKKI.INI and use the debug menu's "Reload Game INI File" to apply your changes immediately, rather than needing to reboot the game.
