# Simple 2000 Series Vol. 33 - The Jet Coaster - Yuuenchi Otsukkurou!

This is the original release of Rollercoaster World which also has a patch in this repository. This one is a bit different though, as the game's main executable is "JD3.ELF", so apply the main patch to this file. The file that looks like the main executable is actually a launcher that isn't needed, so you can safely delete it.
